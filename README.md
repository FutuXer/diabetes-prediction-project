# diabetes-prediction-project
基于统计建模的糖尿病预测与健康风险分析系统
1. 研究领域
医疗健康 - 女性糖尿病风险评估
2. 问题背景
糖尿病是全球重大公共卫生问题，其中女性糖尿病患者面临独特的生理和代谢特征。Pima印第安人群体因遗传和生活方式因素，糖尿病发病率显著高于其他人群，为研究女性糖尿病风险因素提供了宝贵的流行病学数据。早期风险识别和精准评估对预防和延缓疾病进展具有重要临床意义。
3. 核心问题
如何基于常规体检指标，构建可解释的统计模型，对女性糖尿病风险进行量化评估和分类诊断？
4. 具体挑战

数据质量问题：原始数据存在隐藏缺失值（如血压、胰岛素等生理指标出现不合理的零值）
模型可解释性：医疗场景需要模型参数具有明确临床意义，而非"黑盒"预测
风险量化需求：既需要连续的风险评分（回归），又需要明确的患病诊断（分类）
应用落地场景：如何将统计模型转化为实用的临床决策支持工具

5. 研究目标

探索性分析：识别影响女性糖尿病的关键风险因素及其相互关系
回归建模：构建风险评分模型，量化个体患病风险程度（0-100分）
分类建模：建立诊断模型，预测个体是否患病（二分类）
系统开发：开发交互式Web应用，实现个人评估和批量筛查功能

6. 预期价值

临床应用：为社区卫生中心、体检机构提供低成本筛查工具
健康管理：帮助高风险女性及早干预，降低发病率
科研价值：验证统计建模在小样本医疗数据中的有效性
