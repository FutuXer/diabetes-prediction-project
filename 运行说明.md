# 糖尿病预测项目 - 运行说明

## 🚀 快速启动

### 1. 环境配置
```bash
# 安装依赖
pip install -r requirements.txt

# 或使用conda
conda create -n diabetes-prediction python=3.9
conda activate diabetes-prediction
pip install -r requirements.txt
```

### 2. 启动应用
```bash
# 方法1：启动主应用（推荐）
streamlit run app.py

# 方法2：直接启动数据可视化页面
streamlit run pages/4_data-observation.py

# 方法3：启动增强版交互式页面
streamlit run pages/interactive_data_insights.py
```

### 3. 访问应用
启动后，浏览器会自动打开 http://localhost:8501

## 🎯 成员A功能说明

### 核心职责
- **数据可视化与探索性分析（EDA）**
- 生成离线分析图表
- 构建交互式前端界面

### 功能模块

#### 1. 离线分析 (`src/1_visualization.py`)
```bash
# 生成所有离线图表
python src/1_visualization.py
```

输出文件：
- `docs/images/01_overview.png` - 数据概览
- `docs/images/02_distributions.png` - 特征分布
- `docs/images/03_boxplots.png` - 箱线图分析
- `docs/images/04_outcome_comparison.png` - 分组对比
- `docs/images/05_correlation_heatmap.png` - 相关性热力图
- `docs/images/06_pairplot.png` - 散点图矩阵
- `docs/visualization_report.txt` - 分析报告

#### 2. 在线展示 (`pages/4_data-observation.py`)
- **数据概览**：样本统计、分布饼图、描述性统计
- **单变量分析**：直方图、箱线图、统计指标
- **双变量分析**：分组对比、散点图矩阵
- **相关性分析**：热力图、特征重要性
- **风险因素排序**：重要性评估、临床意义

#### 3. 增强版交互式 (`pages/interactive_data_insights.py`)
- **3D可视化**：多维度特征空间探索
- **雷达图对比**：组间/个体对比分析
- **交互式控制**：侧边栏参数调整
- **性能优化**：数据缓存、响应式设计

## 🔧 功能特色

1. **静态图表**（Matplotlib）

## 📱 使用指南

### 数据概览页面
1. 查看数据集基本信息
2. 了解患病率分布
3. 检查数据质量（缺失值、异常值）
4. 查看特征列表和零值检测

### 单变量分析
1. 选择要分析的特征
2. 查看分布直方图和密度曲线
3. 检查异常值情况
4. 了解统计特征（均值、中位数等）

### 双变量分析
1. **单特征对比**：选择特征对比患病组差异
2. **散点图矩阵**：选择多个特征查看相关性

### 相关性分析
1. 查看特征间相关性热力图
2. 发现强相关特征对
3. 了解特征与患病风险的相关性

### 风险因素排序
1. 查看基于相关性的特征重要性
2. 了解临床意义和医学解释
3. 获取关键发现和建议

## 🔍 故障排除

### 常见问题

1. **数据文件找不到**
   - 确保`src/data/diabetes.csv`文件存在
   - 检查文件路径和权限

2. **中文字体显示问题**
   - 已内置字体处理逻辑
   - Windows系统会自动使用Microsoft YaHei

3. **图表显示异常**
   - 检查依赖库版本
   - 清除浏览器缓存后重试

4. **性能问题**
   - 大数据集建议采样分析
   - 使用缓存减少重复计算
